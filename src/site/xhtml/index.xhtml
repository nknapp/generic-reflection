Generic Reflection Utils

This library currently consists only of the class ResolvedGenericType. This class can be used to answer the following question:

Given a class Child<X,Y>:

  1. For a given X and Y what is the direct superclass of Child<X,Y> (including generic parameters)?
  2. For a given X and Y what are the direct superclass implemented by Child<X,Y> (including generic parameters)?
  3. Given Parent<T> a transitive generic superclass or generic interface of Child, what is T?
  4. Is Child<String,Integer> assignable to Parent<Integer>
    5. ?
The class only works in the simple case where type parameters are themselves concrete classes. Generics within type parameters and parameters that are resolved from elsewhere won't work at the moment. Examples can be found in the test case org.knappi.opensource.genericreflection.ResolvedGenericTypeTest.
